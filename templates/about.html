About Open Windows


Welcome to the OpenWindows web application.


This is a free to use web application whose purpose is to assist design teams in the specification of glazing and ventilation for newly built and converted residences in England and Wales in places where noise intrusion is considered to be an issue. Based on average and impulsive noise levels outside the dwelling, which have been determined generally by measurement surveys, the application allows the user to select a specification of glazing which will ensure that the residents are not exposed to excessive noise levels inside a dwelling. In doing so, it allows the design team to take into account the effect of other elements of the building facade, notably the ventilation.


The web application was created by Sulltech Ltd.
