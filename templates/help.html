Help for entry fields

One can enter up to four types of noise levels which the facade is exposed to. The first are energy "average" noise levels over a 16 hour daytime period and an 8 hour night-time period. The second two represent "impulsive" noise. The first for ventilation is assumed to be the 10th highest maximum to occur in a night-time period and the second for ventilation is the highest. Enter either an overall level in dB(A) in the format nn.n or the spectra in tne format nn.n/nn.n/nn.n/nn.n/nn.n where each element of the delimited list Represents the level at each octave band between 125 and 200 Hz.


Enter either a volume in m3 OR area (m2)x height (m) or 3 dimensions (length (m) x width (m) x height (m) )

From drop-down list enter the type of element you wish to include on the facade.

Enter the area of facade of the element or equivalent area for ventilation elements.

If this element is placed on a quieter facade than enter the number of deciBels quiter.

Filter out elements that do not inclide keywords entered here.
